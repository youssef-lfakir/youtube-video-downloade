# 🎬 Simple YouTube Downloader

A lightweight and efficient Python script to download YouTube videos in the best available quality using the `yt-dlp` library.

## 🚀 Features
- Downloads videos in maximum resolution (`format: best`).
- Automatically names the downloaded file based on the YouTube video title.
- Includes basic error handling for broken or invalid URLs.

## 🛠️ Requirements
You need to have Python installed on your machine. Install the required dependency using pip:
```bash
pip install yt-dlp
